# Ebook: Dicionário de Termos Técnicos de Inteligência Artificial para Iniciantes

## 📒 Descrição
Este dicionário de 15 páginas contém os principais termos utilizados em IA para que o iniciante consiga entender os conceitos e aprofundar seus concimentos.

## 🤖 Tecnologias Utilizadas
- IA Generativa **[ChatGPT](https://chat.openai.com)** para roteirização e revisão;
- IA Generativa **[Leonardo AI](https://leonardo.ai)** para criar a arte visual do e-book;
- Ferramenta **Word** para edição.


## 🧐 Processo de Criação
Utilizei o "ChatGPT" para estruturar o dicionário de IA e conter os principais termos utilizados. Para geração de imagens utilizei a ferramenta "Leonardo AI" para o e-book. Para concluir, a montagem final foi feita no Word e conversão do arquivo em pdf.

## 🚀 Resultados
O e-book apresenta visual envolvente e conteúdo resumido porém de grande relevância para quem está inciando os estudos em IA.
